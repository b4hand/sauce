Detailed Description

template<typename Dependency>
class sauce::internal::InitialClause< Dependency >

Base class for initial parts of the fluent binding API.

An initial clause is what all binding sentences begin with, but do not contain enough information to yet create bindings. When they do, a transition to a FinalClause occurs.
