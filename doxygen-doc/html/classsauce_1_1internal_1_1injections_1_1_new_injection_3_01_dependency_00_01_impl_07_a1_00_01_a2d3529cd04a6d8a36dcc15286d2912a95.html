Provide an Iface.

A naked instance pointer is allocated and wrapped in a shared_ptr. It is also given a custom deleter, to dispose of the naked pointer with dispose(Iface *).

Establish that further dependencies do not introduce cycles with ones already accumulated.

This is Tarjan's algorithm using the call stack. When a cycle is detected a CircularDependencyException is thrown.

Reimplemented from sauce::internal::injections::ProvidingInjection< Dependency >.
